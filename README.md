## Overview

An automated pipeline for detecting and grading vertebral fractures in CT scans using the [VerSe19](https://github.com/anjany/verse) public benchmark dataset. The pipeline applies Genant semi-quantitative grading to lumbar vertebrae (L1–L5) by chaining three specialised deep learning models.

Three approaches were developed in sequence, each informed by the failure of the previous:

| Approach | Method | Outcome |
|---|---|---|
| 1 | Random Forest + hand-crafted features | Baseline — 7.85 px corner error, insufficient |
| 2 | CNN cascade (ResNet50 + U-Net) | CNN1 succeeded; CNN2 heatmap regression failed structurally |
| 3 | YOLO three-model pipeline | **Final solution** — median 2.88 px, 96.1% detection rate |

---

## Pipeline

```
CT scan (NIfTI)
    │
    ▼
Sagittal MIP extraction
    │
    ▼
Model 1 — YOLOv8n-seg
Segment all vertebrae, sort top → bottom
    │
    ▼
Model 2 — EfficientNet-B0
Classify which detections are L1–L5
    │
    ▼
Model 3 — YOLOv8n-pose
Predict 4 corners (AS, PS, AI, PI) per labelled vertebra
    │
    ▼
Genant grading
Compute height ratios → Grade 0–3
```

**Corner convention:** `corners[0]` = AS (anterior-superior), `[1]` = PS, `[2]` = AI, `[3]` = PI

## Genant Grading Logic

```python
ha = dist(AS, AI)                              # anterior height
hp = dist(PS, PI)                              # posterior height
hm = dist(midpoint(AS, PS), midpoint(AI, PI)) # mid-body height

ratio = min(ha, hm, hp) / max(ha, hm, hp)

# Grade thresholds (Genant et al., 1993)
if   ratio >= 0.80: grade = 0  # Normal
elif ratio >= 0.75: grade = 1  # Mild
elif ratio >= 0.60: grade = 2  # Moderate
else:               grade = 3  # Severe

# Morphology
if   ha == min(ha, hm, hp): morphology = "Wedge"
elif hm == min(ha, hm, hp): morphology = "Biconcave"
else:                        morphology = "Crush"
```
